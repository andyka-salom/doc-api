ERP Heaven Scent — REST API

Dokumentasi lengkap seluruh endpoint API. Semua endpoint (kecuali auth & Company Profile publik) memerlukan autentikasi via Bearer Token atau Cookie. Gunakan ?search= untuk pencarian teks di semua endpoint GET list.

Base URLhttp://localhost:3000/api
AuthBearer Token / Cookie
FormatJSON
Pagination?page=1&limit=20
SETTING Autentikasi
POST /auth/login Login & dapatkan token ▼
Request Body
{
  "email": "admin@heavenscent.id",  // required
  "password": "rahasia123"          // required, min 6 karakter
}
Response 200
{
  "message": "Login berhasil",
  "token": "eyJhbGciOiJIUzI1...",
  "refresh_token": "eyJhbGciOiJIUzI1...",
  "user": {
    "id": "uuid",
    "email": "admin@heavenscent.id",
    "full_name": "Admin Utama",
    "is_superadmin": true,
    "permissions": ["users.read", "assets.create"]
  }
}
POST /auth/refresh Refresh access token ▼
Token bisa dikirim via cookie atau request body.
Request Body (opsional jika pakai cookie)
{ "refresh_token": "eyJhbGci..." }
Response 200
{ "token": "eyJhbGci...", "refresh_token": "eyJhbGci..." }
POST /auth/logout Logout & invalidasi token ▼
Response 200
{ "message": "Logout berhasil" }
